Questions:

1. JPEG is good for detailed photos with a lot of colors, like portraits or landscapes.

2. GIF is good for simple images with few colors or animations, like icons or logos.

3. PNG is unique because it uses lossless compression and supports transparent background photos.

4. You can resize a photo by using photo editing software to set a new width and height.

5. Resizing in software makes the file smaller, while CSS just shrinks how it looks without reducing file size.
