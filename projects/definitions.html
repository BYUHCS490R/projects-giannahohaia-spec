Internet

Global network that connects computers and devices.

Web

System of websites accessed through browsers.

How are the Internet and Web different?

The Internet is the network, while the web is a service on the network.

ASCII and Unicode

ASCII only encodes 128 characters, while Unicode encodes letters and symbols from most languages .

What is UTF-8

Unicode encoding that uses 1-4 bytes per character.

URL

The address of a resource on the web.

Domain Name

Condensed web address like google.com.

HTTP

Protocol for transferring web data.

HTTPS

Secure, encrypted HTTP.

IP Address

A unique number that identifies a device on a network.

DNS

Translates domain names into IP addresses.

W3C Recommendations

Web approval standards made by the WWW Consortium.

XML, XHTML, HTML, HTML5

XML stores and shares data.

XHTML is a stricter, XML-based version of HTML.

HTML is web page language.

HTML5 is the current version of HTML.
